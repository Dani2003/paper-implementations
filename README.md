# LoRA From Scratch

A clean-room implementation of Microsoft's Low-Rank Adaptation (LoRA) paper in PyTorch.

Instead of relying on PEFT or existing implementations, this notebook rebuilds the core algorithm from scratch to understand how parameter-efficient fine-tuning works internally.

Topics covered

• Low-Rank Decomposition

• Frozen Transformer Weights

• GPT-2 Integration

• Gradient Verification

• Parameter Efficiency

• Training Loop
